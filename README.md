# Alarm System

Ferramentas:

-   PHP
-   Bootstrap
-   JavaScript
-   MySql,

Bibliotecas:

-   PHPMailer

## Telas:

Login

Register

Equipamentos

Alarmes

Alarmes atuados

## Entidades:

Equipamentos:

-   Nome do equipamento;
-   Número de Série;
-   Tipo (Tensão, Corrente e Óleo);
-   Data de cadastro.

Alarmes:

-   Descrição do alarme;
-   Classificação (Urgente, Emergente e Ordinário) ;
-   Equipamento relacionado;
-   Data de cadastro.

Alarmes atuados:

-   Data da entrada;
-   Data da saída;
-   Status do alarme;
-   Descrição do alarme;
-   Descrição do equipamento
